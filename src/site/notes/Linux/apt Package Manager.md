apt is a superset of apt-get and some other tools <https://phoenixnap.com/kb/apt-vs-apt-get>

## Common Commands
- `sudo apt update` - update lists of packages
- `sudo apt upgrade` - upgrade packages
- `sudo apt install [package]` - install a package
- `sudo apt remove [package]` - remove a package
- `dpkg -l` - list installed packages (grep to filter)
