## Vintage Mac Tips 
- Software
	- [Macintosh Garden](https://macintoshgarden.org/) is the best repository of abandonware including classic operating system install discs
- Compression
	- If you have a machine that doesn’t yet have a copy of StuffIt Expander, you can install [this version](https://macintoshgarden.org/apps/stuffit-expander-55) from a disk image
- Disc ripping and burning
	- Disk Utility can rip floppies or CDs to disk images
- Web Browsers: the following are good modernish options
	- Mac OS 9: [Classilla](https://macintoshgarden.org/apps/classilla)
	- Mac OS X 10.4 to 10.6: [PowerFox](https://powerfox.jazzzny.me/)
- Emulation
	- [Infinite Mac](https://infinitemac.org/)
	- Sheep Shaver
	- UTM
- Hardware
	- [EveryMac.com](https://everymac.com/) is a great reference for hardware specs on current and past Apple hardware
	- [eBay](https://www.ebay.com/) is good for variety of machines for sale, but beware overpriced items
	- [ShopGoodwill.com](https://shopgoodwill.com/home) can sometimes have bargains since it gets less traffic
